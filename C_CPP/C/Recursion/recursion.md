## 函数递归

- 程序调用自身的编程技巧称为递归 recursion

```C
最简单的递归函数

int main()
{
    printf("hehe\n");
    main(); // 无限循环造成栈溢出

    return 0;
}

```
### 递归的两个必要条件
- 使用递归函数时，需要提前设定限制条件，当不再满足该限定条件时，结束递归
- 每次递归调用之后越来越接近这个限制条件
- 防止无限递归导致栈溢出

```C
斐波那契数列
1 1 2 3 5 8
a b c 

int a = 1;
int b = 1;
int c = 1;

c = a + b;
a = b;
b = c;
```
